# Sheep - 个人学习工作区

## 学习目标
通过本项目的学习，做到：
1. 掌握 ROS 机器人操作系统的核心概念和编程技能
2. 学会使用 MAVROS 控制无人机
3. 理解并能够复现 `sl_sensor` 项目
4. 为未来的机器人和无人机开发打下坚实基础
